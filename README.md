# 🎬 YouTube Clone

Лёгкий и быстрый клон YouTube на чистом **HTML, CSS и JavaScript** — без фреймворков и без бэкенда. Работает прямо в браузере: локально (просто открыть `index.html`) и на **GitHub Pages**.

🔗 **Демо:** https://zenka737.github.io/youtubeclone/

---

## ✨ Возможности

- 🔍 **Поиск** по названию видео в реальном времени
- 🗂️ **Категории**: Программирование, Дизайн, Музыка, Развлечения, Другое
- ▶️ **Реальное воспроизведение YouTube-видео** через встроенный плеер (iframe) — просто вставьте ссылку на видео
- ➕ **Добавление своих видео** прямо на сайте (название, ссылка, категория, автор)
- ❤️ **Лайки** и 👁️ **просмотры** — сохраняются в `localStorage`, переживают перезагрузку страницы
- 🌙 **Тёмная/светлая тема** с запоминанием выбора и автоопределением системной темы
- 📱 **Адаптивный дизайн** — корректно работает на телефоне, планшете и десктопе
- ⚡ Никаких зависимостей, сборки или сервера — просто открыть и пользоваться

## 🚀 Быстрый старт

### Вариант 1 — открыть локально
```bash
git clone https://github.com/Zenka737/youtubeclone.git
cd youtubeclone
```
Затем просто откройте `index.html` в браузере (двойным кликом или через "Открыть файл").

### Вариант 2 — локальный сервер (рекомендуется)
Некоторые браузеры ограничивают локальные файлы, поэтому лучше поднять простой сервер:
```bash
# Python 3
python3 -m http.server 8000

# или Node.js
npx serve .
```
Откройте http://localhost:8000 в браузере.

### Вариант 3 — GitHub Pages
Проект уже развёрнут и доступен онлайн: https://zenka737.github.io/youtubeclone/

Чтобы задеплоить свою версию: `Settings → Pages → Source: main / (root)`.

## 🎥 Как добавить своё видео

1. Нажмите кнопку **«➕ Добавить видео»** в шапке сайта
2. Укажите название, категорию и (по желанию) ссылку на YouTube
3. Нажмите **«Опубликовать»** — видео появится в общей ленте с реальным плеером

Видео сохраняются в вашем браузере (`localStorage`) — они не отправляются на сервер и видны только вам на этом устройстве.
